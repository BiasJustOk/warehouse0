# 简易TXT文件编辑器（C语言）
这是一个基于C语言开发的极简版TXT文件管理工具，支持TXT文件的创建、内容追加编辑、读取、删除功能，使用Visual Studio开发。

##  开发环境
- 编译器/IDE：Microsoft Visual Studio（任意版本，如VS2019/2022）
- 编程语言：C语言（ANSI C标准）
- 运行平台：Windows系统

##  核心功能
| 功能编号 | 功能描述 | 细节说明 |
|----------|----------|----------|
| 1        | 创建TXT文件 | 输入文件名，若文件已存在可选择覆盖；输入内容（空行结束输入）并保存 |
| 2        | 编辑TXT文件 | 仅支持**追加内容**到已有TXT文件末尾（空行结束输入） |
| 3        | 读取TXT文件 | 输入文件名，打印文件内的全部内容 |
| 4        | 删除TXT文件 | 输入文件名，校验文件存在后删除 |
| 5        | 退出系统 | 关闭程序 |

##  编译&运行方法（Visual Studio）
1. 打开Visual Studio，点击「创建新项目」→ 选择「空项目」→ 命名项目并选择保存路径 → 点击「创建」；
2. 在项目的「源文件」目录下，右键→「添加」→「现有项」，将你的C语言源码文件（one.c）添加到项目中；
3. 点击顶部菜单栏「生成」→「生成解决方案」（或按F7），等待编译完成；
4. 编译成功后，点击「调试」→「开始执行（不调试）」（或按Ctrl+F5），即可运行程序。

##  使用说明
1. 运行程序后，会显示功能菜单；
2. 输入数字1-5选择对应功能，按回车确认；
3. 按提示输入文件名/文件内容即可完成操作（文件名无需加.txt后缀，程序会自动处理）；
4. 输入内容时，若需结束输入，直接按「回车」输入空行即可。

##  注意事项
1. 程序仅支持Windows系统，暂不兼容Linux/Mac；
2. 编辑文件仅支持「追加内容」，暂不支持修改/删除已有内容；
3. 删除文件时请谨慎，删除后无法恢复；
4. 文件名请勿包含特殊字符（如/、\、:、*、?等），否则可能操作失败。
